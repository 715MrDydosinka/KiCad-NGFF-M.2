# Kicad NGFF M.2 Symbols+Footprints

A KiCad library providing schematic symbols and PCB footprints for various NGFF (M.2) connectors.

## Contents

### Symbols
- `Socket1_Key-A_DP`
- `Socket1_Key-E`
- `Socket1_Keys-AE`
- `Socket2_Key-B_PCIe-WWAN`
- `Socket2_Key-B_PCIe_SSD`
- `Socket2_Key-B_SATA_SSD`
- `Socket2_Key-B_SSIC-WWAN`
- `Socket2_Key-B_USB3-WWAN`
- `Socket3_Key-M_PCIe_SSD`
- `Socket3_Key-M_SATA_SSD`

### Footprints
- `NGFF_Key-A`
- `NGFF_Key-E`
- `NGFF_Key-AE`
- `NGFF_Key-B`
- `NGFF_Key-M`
- `NGFF_Key-BM`
